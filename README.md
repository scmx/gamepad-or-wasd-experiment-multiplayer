# Two-player game prototype with Gamepad and WASD controls

Compete two players against two bots to collect 100 coins together first.

Pay 3 coins to place a poison that will drain coins from the bots.

Avoid colliding with the bots, since they'll drain your coins.

https://scmx.github.io/gamepad-or-wasd-experiment-multiplayer/
